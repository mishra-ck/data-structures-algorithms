Binary Search Tree 
---
Binary Search Tree (BST) is a specific type of binary tree that maintains a strict ordering of its elements.
While every BST is a binary tree, not every binary tree is a BST.
---
### The core rule of a BST is the Ordering Property

For any given node:
1. All nodes in its left subtree must have values less than the node’s value. 
2. All nodes in its right subtree must have values greater than the node’s value.
