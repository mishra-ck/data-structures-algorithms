Tree Data Structure 
---
Tree is a hierarchical, non-linear data structure having nodes connected by edges.
Unlike arrays or linked-list, tree represent parent-child relationships.
---
### Key Concepts
```
✅ Node: Basic unit of storing data + reference to children
✅ Root: The topmost node
✅ Leaf: A leaf with no children
✅ Edge: Link between parent and child
✅ Height: Longest path from root to a leaf
✅ Depth: Distance from root to that node
✅ Degree: number of children a node has
✅ Subtree: A node and all its subtree
✅ Ancestor: Any node on the path from root to a node
✅ Sibling: Nodes sharing the same parent
```
