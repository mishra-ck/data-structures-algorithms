Universal DSA Patterns 
---

| # | Pattern | Key Signal in Problem | Structures Involved |
|---|---|---|---|
| 1 | **Two Pointers** | Sorted array, pair/triplet sum, palindrome | Array, String |
| 2 | **Sliding Window** | Subarray/substring of size k, max/min range | Array, String, Deque |
| 3 | **Fast & Slow Pointers** | Cycle detection, middle of list | LinkedList |
| 4 | **Merge Intervals** | Overlapping intervals, scheduling | Array, Sort |
| 5 | **Cyclic Sort** | Numbers in range [1..n], missing/duplicate | Array |
| 6 | **In-Place Reversal** | Reverse sublist, rotate list | LinkedList |
| 7 | **BFS / Level Order** | Shortest path, level traversal | Graph, Tree, Queue |
| 8 | **DFS / Backtracking** | All paths, permutations, combinations | Graph, Tree, Stack |
| 9 | **Binary Search** | Sorted input, "find minimum/maximum satisfying" | Array, BST |
| 10 | **Top K Elements** | Kth largest/smallest, frequent | Heap, QuickSelect |
| 11 | **Two Heaps** | Median, balanced partition | Min-Heap + Max-Heap |
| 12 | **Subsets** | All combinations/subsets/permutations | Array, Backtracking |
| 13 | **Dynamic Programming** | Optimal substructure, overlapping subproblems | Array, Matrix, String |
| 14 | **Monotonic Stack/Queue** | Next greater/smaller element, max in window | Stack, Deque |
